## 측정 원칙
- **참조물**: 동일 측정 회기에선 항상 같은 참조물(신용카드 권장) 사용.
- **실측 기준**: 자·줄자. 가능하면 mm 단위 척.
- **반복**: 동일 대상 3회 측정 후 평균을 시트에 기록.
- **이상치**: 명백한 보정 오류(4점이 카드 모서리를 벗어남)는 제외하고 재측정 — 비고에 사유 기록.
- **회귀 방지**: 새 시트를 만들 때 이전 시트의 1·5·9번 대상은 동일 조건으로 재측정.
