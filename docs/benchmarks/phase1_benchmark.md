# Phase 1 벤치마크 — 사진 측정(참조 객체)

> design.md §1.3 성공 기준: 사진 모드 평균 오차 ≤ 5%, 콜드 스타트 ≤ 30초.
> AR 모드는 W2/W3 보류로 1차 게이트에서 제외 (W7~W10 이후 재측정).

## 측정 환경
- 측정자:
- 일자:
- 디바이스/에뮬레이터:
- 앱 버전: 1.0.0+1 (release/debug)
- 조명: (예: 형광등, 직접광, 자연광 — 균일/그림자 유무 기록)
- 카메라 거리: 단말 ~ 대상 약 30~50 cm

## 측정 절차 (대상 1개당)
1. 신용카드를 대상 옆 동일 평면에 두고 위에서 수직으로 촬영.
2. 앱에서 사진 — 참조 객체 → 4점 보정(카드 모서리에 정렬) → 측정 종류 선택 → 점 입력 → 측정.
3. 결과(mm)를 시트의 `측정값(mm)` 열에 기록.
4. 자·줄자로 실측(mm)을 같은 열의 `실측값(mm)`에 기록.
5. 오차(%) = `|측정 - 실측| / 실측 * 100`. 시트 함수 또는 수동 계산.

## 콜드 스타트 측정 절차
1. 앱을 멀티태스킹에서 완전히 종료 (스와이프 닫기).
2. 스톱워치 준비.
3. 앱 아이콘 탭 ↔ 스톱워치 시작.
4. 모드 선택 → 사진 — 참조 객체 → 갤러리 → 이미지 선택 → 4점 보정 → 두 점 탭 → 측정 → 결과 화면 표시.
5. 결과 화면에 측정값이 표시되는 순간 스톱워치 중지.
6. 3회 측정 후 평균 (`s1, s2, s3, avg`):  ____ s.

## 회귀 방지 메모
- 오차 5% 초과 항목은 비고에 원인 가설(보정 부정확/그림자/카메라 거리) 기록.
- 다음 단계(W7 LiDAR, W8 동전 자동 감지 등) 진입 전 본 시트 결과를 컨텍스트 노트에 인용.
- 같은 시트를 W10·W16 종료 시점에 다시 측정해 누적 회귀 확인.
